This is a (RAG) chatbot built using Next.js, Google Gemini, and Pinecone.Used  Axios, Cheerio.scrap raw text from https://resend.com/sitemap.xml.The sitemap is fetched using Axios and cheerio for raw text extraction.The pages links and text is saved in resend_data.json. Utilizes Google Gemini text-embedding-004 to summarize content and generate 5 Q&A pairs per page.use timers to handel the gemini free triel to saved these pages in page_qa.json. fs-extra, dotenv, xml2js these Utilities are used to handle the complex tasks of web scraping, data processing, and security..There is two API KEYS GOOGLE_API_KEY,PINECONE_API_KEY.Generates embeddings and upserts them into Pinecone.
